# Estructuras de Datos – Árboles

Este repositorio contiene implementaciones y ejercicios prácticos relacionados con estructuras de datos basadas en árboles, desarrollados como parte de la asignatura de Estructuras de Datos.

## Contenido del Repositorio

### Implementaciones
- Árbol General
- Árbol Binario

Ambas estructuras incluyen las operaciones básicas necesarias para trabajar con árboles.

### Ejercicios

1. Árbol Genealógico (Árbol General)  
Simulación de un árbol genealógico donde cada nodo representa una persona.  
Permite realizar operaciones como:
- Buscar una persona en el árbol
- Mostrar antepasados
- Mostrar descendientes
- Eliminar una rama familiar
- Calcular la generación más profunda (altura del árbol)
- Recorrer el árbol por generaciones (BFS)

2. Simulación de Torneo (Árbol Binario)  
Modelo de un torneo eliminatorio utilizando un árbol binario.  
El sistema simula:
- Cuartos de final
- Semifinales
- Final

Cada nodo representa un partido entre dos equipos y el ganador se determina de forma aleatoria.

## Tecnologías
- Java

## Objetivo
El objetivo de este repositorio es practicar la implementación y el uso de estructuras de datos basadas en árboles, así como aplicar diferentes tipos de recorridos y operaciones sobre estas estructuras.
